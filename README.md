## 📖 Description

AppHeaven is a modern and responsive App Store web application where
users can discover applications, check app details, install apps, and
manage their installed applications. This project demonstrates React
fundamentals, routing, state management, LocalStorage integration,
search functionality, sorting systems, and data visualization.

------------------------------------------------------------------------

## 🚀 Key Features

### 🏠 Home Page

-   Responsive modern banner section
-   App Store & Play Store redirect buttons
-   Statistics section
-   Top apps showcase
-   Show All button navigation

### 📱 All Apps Page

-   Live search functionality
-   Case-insensitive search
-   Dynamic app counter
-   No Apps Found message
-   Responsive grid layout

### 📊 App Details Page

-   Detailed app information
-   Install button with state control
-   Success toast notification
-   Review statistics chart using Recharts
-   App description section

### 💾 Installation System

-   Install apps using LocalStorage
-   Prevent duplicate installations
-   My Installation page
-   Uninstall feature
-   Toast notification system
-   Sort installed apps by downloads

### ⚡ Additional Features

-   Custom 404 error page
-   Loading animation
-   Active route highlighting
-   Responsive design
-   Route reload support

## ⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/zunaid1/app-heaven-app.git

Navigate to project folder:

cd app-heaven-app

Install dependencies:

npm install

Run project:

npm run dev
